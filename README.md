# Math Adventure Preschool VN

Ứng dụng demo học toán mầm non cho trẻ 4–5 tuổi, tối ưu cho trải nghiệm trực quan, ít chữ, nhiều tương tác hình ảnh.

## Tính năng
- Trang chủ giới thiệu sản phẩm
- Chọn hồ sơ bé
- Dashboard theo tiến trình
- Màn game học toán trực quan
- Góc phụ huynh với tóm tắt AI mô phỏng
- Cài đặt giao diện và trải nghiệm
- Lưu tiến trình bằng `localStorage`

## Công nghệ
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## Chạy dự án
```bash
npm install
npm run dev
```

## Build production
```bash
npm run build
```

## Ghi chú
Đây là bản demo UI/UX và logic gợi ý học tập. Các dữ liệu AI hiện là mô phỏng, có thể thay bằng API thật sau này.
